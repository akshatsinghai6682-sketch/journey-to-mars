Journey to Mars

A storytelling website built for Frontend Odyssey (IIT Patna).

Concept
This project takes users on an immersive journey from Earth to Mars through smooth scrolling and animated sections.

Features
- Smooth scrolling navigation
- Interactive sections (Origin → Launch → Space → Mars)
- Animated UI and glowing effects
- Scroll progress bar
- Responsive design

Tech Used
- HTML
- CSS
- JavaScript
